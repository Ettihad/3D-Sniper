# 🎯 3D Sniper

It is a 3D interactive game which uses camera detection. No download. No installation. You can play it using only your fingers.

Experience the deployment instantly without any manual script installations or binary downloads:

## 🎮 How to Play

1. **Launch the Game:** Open the live deployment link in any modern desktop web browser.
2. **Configure Permissions:** Grant camera access if prompted by the interface to enable integrated tracking mechanics.
3. **Aim & Target:** Use your index to aim and shoot by pinching your index and thumb.
4. **Blast Targets:** Interact dynamically with user interface systems to eliminate active targets and increase your score.
